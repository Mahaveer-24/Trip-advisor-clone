#Page Description
-This project is a responsive travel review website inspired by Tripadvisor. 
-It is built entirely using HTML and CSS, focusing on recreating a modern travel discovery interface with structured layouts, rating components, and responsive design.
-The goal of this project is to demonstrate strong frontend fundamentals, including semantic HTML, CSS Flexbox/Grid, and mobile-friendly design without using JavaScript or backend technologies.

#Project Purpose
This project was created to: 
-Strengthen HTML & CSS fundamentals. 
-Practice layout design and alignment. 
-Improve styling, positioning, and hover effects. 
-Recreate a real-world website interface for learning purposes.
